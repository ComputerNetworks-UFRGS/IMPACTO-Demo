{% trans "Risk Analysis Report" %}

{% trans "The Risk Analysis Report provides a evaluation of the company's exposure to various cyber threats. It combines data on regional and sector-specific risks with the company's own resilience measures to present a risk profile." %}

{% trans "Risk Analysis Table" %}

{% trans "This table displays the risk levels associated with different types of cyber threats, categorized into Region Risk, Sector Risk, and Company Resilience. Each threat (Malware, Phishing and DDoS) is evaluated across these dimensions to provide a understanding of the risk landscape." %}

{% trans "The risk levels are visualized using risk bars, which are color-coded indicators showing the degree of risk from 'Excellent' to 'Poor'. For Region Risk and Sector Risk, the bars are inverted, indicating that lower values represent lower risk. For Company Resilience, higher values indicate stronger resilience against threats." %}

{% trans "Interpreting the Risk Bars" %}

  • {% trans "Region Risk and Sector Risk: The bars indicate the inherent risk associated with the company's geographical region and industry sector, respectively. A bar leaning towards 'Poor' signifies a higher inherent risk in that category." %}
  • {% trans "Company Resilience: This bar reflects the company's own measures to withstand cyber threats. A bar towards 'Excellent' indicates strong resilience due to effective security measures." %}

{% trans "Risk Score Visualization" %}

{% trans "This section features a displat of the risk scores for each threat type, alongside a final consolidated risk score. The colors and values indicate the level of risk, ranging from 'Healthy' to 'Critical'." %}

{% trans "Understanding the Risk Indicators" %}

  • {% trans "Color Score Indicators: Three circular indicators show the risk score for each threat type (Malware, Phishing, DDoS). The score ranges from 0 to 10, with higher scores indicating higher risk." %}
  • {% trans "Final Score Display: Below the color score system, a score card shows the company's overall risk assessment, combining all threat evaluations into a single metric." %}
  • {% trans "Color Scale: The indicators use a consistent color scheme to represent risk levels:" %}
    • {% trans "Red (Critical): Risk score >= 8.0" %}
    • {% trans "Orange (Dangerous): Risk score >= 6.0" %}
    • {% trans "Yellow (Alert): Risk score >= 4.0" %}
    • {% trans "Green (Normal): Risk score >= 2.0" %}
    • {% trans "Blue (Healthy): Risk score < 2.0" %}

{% trans "Economic Planning" %}

{% trans "The Economic Planning screen offers simulations and analyses to optimize cybersecurity investments and assess their economic impacts. This section includes interactive tables and charts that display investment scenarios and their effects on economic metrics." %}

{% trans "Understanding the Terms" %}

  • {% trans "Status" %} - {% trans "Indicates whether the investment amount is within the acceptable investment window." %}
  • {% trans "Type" %} - {% trans "Indicates the type of cyberattack, such as Malware, Phishing, or DDoS." %}
  • {% trans "Damage" %} - {% trans "Represents the expected monetary loss for a particular type of attack. Customizable through Impacts modal." %}
  • {% trans "Probability" %} - {% trans "Shows the likelihood of a successful attack of a particular type." %}
  • {% trans "Investment" %} - {% trans "The amount currently invested in cybersecurity measures for the asset against a certain type of attack." %}
  • {% trans "Optimal Investment" %} - {% trans "The recommended investment level for maximizing cost-benefit." %}
  • {% trans "Acceptable Investment" %} - {% trans "Ihe investment range where efficiency approaches optimal levels." %}
  • {% trans "EBIS Current" %} - {% trans "The current Economic Benefit Impact Score based on existing investments and its effectiveness on reducing damage." %}
  • {% trans "EBIS Optimal" %} - {% trans "The Economic Benefit Impact Score calculated after applying the optimal investment levels and its effectiveness on reducing damage." %}
  • {% trans "ENBIS Current" %} - {% trans "The current Economic Net Benefit Impact Score based on existing investments." %}
  • {% trans "ENBIS Optimal" %} - {% trans "The Economic Net Benefit Impact Score calculated after applying the optimal investment levels." %}
  • {% trans "Efficiency" %} - {% trans "The current score of the investment compared to the optimal investment" %}
  • {% trans "Adjustment" %} - {% trans "The required adjustment to reach the optimal investment level." %}
  • {% trans "Profit" %} - {% trans "The expected profit after making the investment adjustments." %}

{% trans "In addition to the previously mentioned attacks, the 'General' line provides an overview that aggregates data from the other attacks. These values are calculated using data from the aforementioned attack types, offering a simplified view of the asset's cybersecurity investment status. All calculations utilize an alpha value of 0.001." %}

{% trans "Impacts modal" %}

{% trans "This tool is responsible for configuring the 'Damage' parameters, enabling the user to set both direct and indirect damage levels for the respective asset. Each can be categorized as Low, Medium, or High, based on the potential economic loss associated with various types of attacks affecting that asset." %}

  • {% trans "Direct" %} - {% trans "Represents the immediate and easily attributable damage that results directly from the attack. Examples include data loss, service disruption, recovery costs, and risks to data integrity." %}
  • {% trans "Indirect" %} - {% trans "Refers to secondary and long-term consequences that arise as a result of the attack. Examples include reputation loss, long-term revenue decline, legal and regulatory penalties, and increased operational costs." %}

{% trans "The Economic Planning screen conducts simulations using the Gordon-Loeb model to determine the optimal investment levels for cybersecurity measures.
